Study Bible

In Alkitab you can easily read or study the bible. The main bible toolbar allows you to easily explore the bible with ease.

The following table lists the toolbar buttons and their descriptions.

Button Name Description
Back Go back to the previous page in history . It is similar to web browser history features.
Forward Go Forward to the next page in history. It is similar to web browser history features.
Refresh Refresh the current view.
Previous Go to previous passages. It will be active when the displayed passages is span more than 1 page eg: result of searching
Next Go to next passages. It will be active when the displayed passages is span more than 1 page eg: result of searching
Expand 1 Expand by 1 verse. It is useful to see the context of the verses by displaying the previous and next verses surrounding the current viewed verses
Expand 5 Expand by 1 verse. It is useful to see the context of the verses by displaying the previous and next verses surrounding the current viewed verses
View Options It is the view options (not persisted, only for current tab). For global options (persisted options) and apply to all newly opened tab use Menu-Tools-Options instead.

*Checkbox compare, only applicable when displaying parallel view and in the same language eg: english and english (KJV and ESV)
