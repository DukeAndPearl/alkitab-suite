Overview

Alkitab Bible Study is free desktop based Bible Study tool. It features rich and user friendly Bible study tools. It support parallel view, commentaries, dictionaries, and daily devotions. It also come with powerful search capability. Alkitab is an alternative front end for JSword Crosswire project.

Alkitab Bible Study has been placed into the public domain.

Features

  • View Single Books or Bible
  • View Parallel Books or Bible
  • Cross Reference and Search
  • Commentaries
  • Dictionaries
  • Daily Devotions
  • Written entirely in Java
