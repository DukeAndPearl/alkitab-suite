What is it?

Alkitab Bible Study is free desktop based Bible Study tool. It features rich and user friendly Bible study tools. It support parallel view and powerful search capability. Alkitab is a front end to the sword project which is available for Windows, Unix/Linux, etc.

Alkitab requires Java 1.5 or higher (http://www.java.com/getjava).
As a cross platform application, Alkitab runs on any platform that support Java 1.5
eg: Microsoft Windows, Unix or Linux, etc.

The latest news about Alkitab Bible Study can be found on http://www.kiyut.com

Note: You need to have Java 1.5 or higher, for installation and use.
You can download from Sun Java site (http://www.java.com/getjava)

Installation and Run

To install the product, please do the following:

Mac OSX: You can try the Mac OSX installer (experimental) which is available at http://www.kiyut.com/products/download.html or you can follow the instruction below.

  • unzip the file
  • on Microsoft Windows double click setup.exe or setup.jar
    on Linux/Unix run setup, if necessary chmod +x setup to make it executable
    on Mac OSX run the Java setup by clicking on the setup.jar
    other platform type "java -jar setup.jar"
  • follow the on screen instruction

Important: uninstall or remove the previous version if exist, simply delete the folder will be fine. Or you need to install in different folder.


To run the product after install:

  • on Microsoft Windows just run Alkitab from Desktop Shortcut or Start Menu - All Programs - Kiyut - Alkitab
    on Mac OSX
    • open the Terminal program (inside Applications folder - Utilities)
    • type "cd [INSTALL-DIR]/bin/" eg: cd /home/user-name/Kiyut/Alkitab/bin/
    • type "./alkitab"
    • or you can just run Alkitab by double click the file alkitab at "[INSTALL-DIR]/bin/alkitab" eg: /Applications/Kiyut/Alkitab/bin/alkitab and choose open with terminal
    on other platform run [INSTALL-DIR]/bin/alkitab, if necessary chmod +x alkitab to make it executable
  • enjoy the product
